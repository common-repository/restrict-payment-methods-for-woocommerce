== Description ==

**WooCommerce Restrict Payment Methods** is make to restirication to particular total or another part that is useful for woocommerce checkout.

= Features =

<ul>
<li>One Click installation</li>
<li>Setup Multiple Rules for restication</li>
<li>Easy To use</li>
<li>Enable/Disable Payment Gateways</li>
<li>Order Total by Restrict</li>
<li>Shipping Method by Restrict</li>
<li>Shipping Class by Restrict</li>
<li>Products by Restrict</li>
<li>Products Qty by Restrict</li>
</ul>
